Personal Library

A user front-end for the Free Code Camp Personal Library challenge

This Repl contains my solution to the challenge, using HTML5, CSS3, Node.js, Express, and MongoDB

Use this form to test sending a POST request to the database. Type the title of a book to be added to the library database. You should receive a response from the server with the title and an id.

Use this form to test sending a POST request to the database. Type the id of a book in the database and a comment about that book. You should receive a response from the server with the title, an id, and an array of comments.
